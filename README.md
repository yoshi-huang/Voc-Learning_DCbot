# Voc Learning - 英文單字學習 Discord Bot

> 這是一個高中生自學詞向量並應用到英文單字學習的專案。  
> 動機源於學測前發現自己單字能力不足，透過短時間開發，建立每日單字學習習慣，並從中學會使用 Word2Vec 與 Discord Bot。

---

## 一、動機與背景

在學測前四個月，我發現模考成績顯示：

- 我的英文單字能力是短板  
- 若不加強，很可能影響總分

當時，我剛自學到 **詞向量 (Word2Vec) 與潛空間概念**，突然想到：

> 能不能把這個概念應用到單字學習，建立一個能自動推送單字、並找出語意相近單字的系統？

由於鄰近段考，時間極短，我決定用 **極簡設計快速實作**，整個專案耗時只有 **8 小時**。  
雖然短暫，但後續四個月對我英文學習幫助巨大，最終在學測英文取得超出預期的分數。

這段經驗讓我明白：  

> 即使時間有限，只要專注於核心需求與學習目標，也能創造實際影響。

---

## 二、學習與開發過程

在這個專案中，我的學習過程分為幾個階段：

### 1. 問題定義
- 確定主要需求：每日推送單字 + 語意相似單字  
- 設定可行目標：支援 7000 單字分級學習

### 2. 技術探索
- 了解 **Word2Vec** 預訓練模型的載入與向量操作  
- 學習 Discord Bot 的基本運作與訊息推送

### 3. 系統設計
- 先設計單字推送流程，確保每日能自動更新  
- 建立簡單的相似度計算邏輯  
- 調整單字難度分布與每日推送數量  

### 4. 快速實作
- 使用 Python 與 Discord.py 實作 Bot  
- 測試每日自動推送單字、手動觸發功能  
- 小幅優化相似單字顯示與排程

### 5. 迭代與學測應用
- 在後續四個月使用 Bot 自學單字  
- 建立單字聯想網絡，增強記憶與理解  
- 成功提升英文單字能力，應用於學測取得理想成績

---

## 三、學習心得

這個小專案雖然短暫，但收穫非常明顯：

1. **短時間專注於核心需求**，能快速完成原型並產生實際效益  
2. **將理論概念落地**：詞向量不再只是抽象模型，而能直接支援學習  
3. **建立自動化學習習慣**：每日推送單字幫助我持續累積  
4. **快速學習新工具**：Discord Bot 與程式調用大模型的經驗  
5. **自主學習與問題解決能力提升**：從需求、設計、實作到驗證，完整經歷一個小型專案循環

---

## 四、專案簡介

雖然技術細節不是重點，但概覽如下：

- **核心功能**：每日推送單字、手動查詢相似單字  
- **資料來源**：7000 單字分級列表  
- **技術概念**：Word2Vec 預訓練模型、餘弦相似度  
- **工具**：Python + Discord Bot  

這部分更多是 **支撐動機與學習歷程的工具**，而不是專案本身的焦點。

---

## 五、應用與影響

- **考試準備**：建立每日自動單字學習習慣  
- **單字聯想**：透過語意相似度強化記憶  
- **自主學習能力**：從概念學習到程式實作  
- **短時間專案實踐**：證明快速專注開發也能帶來長期效益
